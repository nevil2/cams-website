# Technologies




**Development is focused on the following technologies**:
1. [![alt b](typescript-logo.svg) Typescript](https://www.typescriptlang.org/) as the programming language.
   <br>&nbsp;&nbsp;&nbsp;&nbsp;Typescript is JavaScript with types. Types speeds up development, improves reliability and facilitates maintenance.
2. ![image alt b](svelte-logo.svg) [Svelte](https://svelte.dev/) / [![alt h](svelte-kit-logo.svg)](https://kit.svelte.dev/) as the front-end framework which bring reactivity to **HTML**, **CSS** and **JavaScript**.
   <br>&nbsp;&nbsp;&nbsp;&nbsp;Svelte is easier to learn, more concise and performant than the above alternatives .
3. [![alt h](tailwindcss-logo.svg)](https://tailwindcss.com/) for styling.
   <br>&nbsp;&nbsp;&nbsp;&nbsp;Tailwind is very widely used to simplify writing CSS.
4. [![alt h](strapi-logo.svg)](https://strapi.io/) as the backend server for club websites.
   <br>&nbsp;&nbsp;&nbsp;&nbsp;Strapi is the leading open-source headless CMS.  It can be customised to fulfil other server functions.
5. [![image alt b](hono.svg)Hono](https://hono.dev/) as an [![alt h](express.svg)](https://expressjs.com/) replacement for the backend server for the routes database and other bespoke applications.
   <br>&nbsp;&nbsp;&nbsp;&nbsp;Hono provides a light-weight, fast server which can be deployed in a variety of runtimes (not just [![alt h](nodejs-logo.svg)](https://nodejs.org/en/about/))
6. [![alt b](sqlite_logo.svg)](https://www.sqlite.org/) for local SQL databases.
7. [![alt b](postgresql-logo.svg)](https://www.postgresql.org/) where a Managed Database Service is used.

**Standardising technologies facilitates development, maintenance and code reuse.**

Some applications were intially built using a range of other technologies including:
- [![image alt h](vue-logo.svg)VueJS](https://vuejs.org/) / [![alt h](nuxt-logo.svg)](https://nuxt.com/) / [![alt h](express.svg)](https://expressjs.com/) for the core routes database and topoeditor
- ![image alt h](react-logo.svg) [React](https://reactjs.org/) for the SMC full routes site
- [![alt h](python-logo.svg)](https://www.python.org/) / [![alt h](fastapi-logo.svg) FastAPI](https://fastapi.tiangolo.com/) / [![alt h](pydantic-logo.svg) Pydantic](https://docs.pydantic.dev/) /
[![alt h](sqlalchemy-logo.png)](https://www.sqlalchemy.org/) for bespoke Munros and Himalayan Index servers

The VueJS and React have already been converted to Svelte / Hono, and during 2025 the python servers will be ported to Hono.
